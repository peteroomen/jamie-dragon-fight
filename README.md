# Knight vs Ice Dragon 🐉

A browser RPG battle game built by Peter & Jamie.

Fight waves of ice dragons, earn gold, and upgrade your knight in the shop between rounds.

## Controls

| Key | Action |
|-----|--------|
| R | Sword Slash |
| S | Shield Bash |
| D | POWER STRIKE (5s cooldown) |
| F | Fire Arrow |
| G | Magic Blast |
| J | LIGHTNING BOLT (6s cooldown) |
| H | Use Health Potion |
| B | Use Dragon Bomb |

## Run it

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
